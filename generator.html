Generator Link Undangan

Tempel daftar nama tamu (satu nama per baris), lalu klik Generate. Kamu akan dapat: (1) kode unik + link siap kirim per tamu, dan (2) baris siap tempel ke Google Sheet kamu.

NamaLink
Siap tempel ke Google Sheet (kolom: code, name) 

    

      Cara pakai: buka Google Sheet kamu → pastikan kolom A = code, kolom B = name →
      tempel hasil di atas mulai baris ke-2 (di bawah header) → File → Share → Publish to web →
      pilih sheet ini → format CSV → copy link-nya → tempel ke CONFIG.SHEET_CSV_URL di file index.html.
